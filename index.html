A commercial alternative to QKD: filling USB sticks and hard disks with random bits
Jun 2017 - Sep 2017

Summer UROP (Undergraduate Research Opportunity) in Cryptography. Supervisor: Professor Kenny Paterson.

Completed research into good sources of randomness, specifically in a cryptographically secure setting for generation of good key material. Created a software solution to use these good sources of randomness to fill hard disks with key material. This key material was secured under a cryptographic scheme in order to preserve security in the case the disk is in the hands of an attacker. An API was also created to allow these random bits to be consumed as keying material, example uses of this API were in an extension to the WireGuard protocol and a simple one time pad encryptor.
